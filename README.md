# 💱 Modern Currency Converter

Ek modern, responsive, aur fast Currency Converter web application jo real-time exchange rates provide karti hai. Isse HTML, CSS, aur JavaScript ke saath banaya gaya hai.

## ✨ Features

* **Real-time Rates:** Fawaz Ahmed ki Currency API ka istemal karte hue bilkul latest rates.
* **150+ Currencies:** Duniya bhar ki currencies ka support.
* **Flag Integration:** Currency select karte hi Flags automatically update hote hain (FlagsAPI).
* **Fully Responsive:** Mobile, Tablet, aur Desktop par perfectly chalta hai.
* **Modern UI/UX:** Glassmorphism aur clean animations ke saath behtareen design.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure ke liye.
* **CSS3:** Custom properties, Flexbox, aur Animations ke liye.
* **JavaScript (ES6+):** Fetch API aur DOM manipulation ke liye.
* **API:** [Currency API](https://github.com/fawazahmed0/currency-api)

## ⚙️ How to Run Locally

1.  Is repository ko clone karein:
    ```bash

    ```
2.  Project folder mein jayein:
    ```bash
    cd currency-converter
    ```
3.  `index.html` file ko kisi bhi browser mein open karein.
